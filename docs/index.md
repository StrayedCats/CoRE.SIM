# CoRE.SIM

![](coresim_top.png)

CoRE.SIMはCoRE-1向けの最高のシーンシミュレータです。CoRE.SIMは、tier4/AWSIMからフォークされたプロジェクトです。

## 特徴

- シミュレーターコンポーネントが含まれています（ロボット、センサー、環境、ROS2など）。
- Windows10/11、Ubuntu 22.04、Ubuntu 20.04のサポート
- ROS2ネイティブ通信
- オープンソースソフトウェア
- Unityゲームエンジンで作成されました。
